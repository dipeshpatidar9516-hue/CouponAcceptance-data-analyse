# Coupon Acceptance Analysis Project

##  Project Overview

This project analyzes customer coupon acceptance behavior using SQL, Python, and Power BI.
The goal is to help marketing teams understand which customers are more likely to accept coupons and how to improve campaign performance.

This project is designed as a real-world portfolio project for data analyst freelancing and job applications.

---

##  Business Objectives

* Identify which customer segments accept coupons most
* Analyze impact of weather, income, age, and destination
* Understand which coupon types perform best
* Recommend strategies to improve marketing ROI

---

##  Dataset Description

The dataset contains customer driving and coupon information with columns like:

* destination
* passenger
* weather
* temperature
* coupon
* expiration
* gender
* age
* maritalStatus
* income
* occupation
* car
* restaurant & coffee visit history
* travel time to coupon
* Accept (Target variable)
